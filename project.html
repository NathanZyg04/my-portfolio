Macro Tracking iOS App

This is a Meal/Macro tracking app developed for iOS using Swift and SwiftUI. It allows users to track their daily calorie and macro intake with an intuitive interface.

Allows adding of food items to a database, adding recipes and templates, logging foods for a day, caluclating total daily macros, saving previous days' data

Currently working on this. Things I want to add is a statistical break down with a easy to read view of graphs and trends of consumption of macros

Also would like to add a scan feature for nutritional labels to make it easier to enter food items

Technologies used: Swift, SwiftUI

Check out the GitHub repository: Macro Tracking App Repo
